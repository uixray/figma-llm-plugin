Data Substitution

Replace placeholder text in your designs with real data using presets.

  1. Go to Data tab
  2. Create a preset with field names (e.g., name, email, phone)
  3. Add groups with values for each field
  4. Select frames/components in Figma
  5. Click Apply to Selection

How It Works

The plugin matches text layer names with field names and replaces their content:

  • Layer named "email" â†’ gets "john@example.com"
  • Layer named "name_email" â†’ gets "John Doe, john@example.com" (multi-value)

Import/Export

Share presets with your team:

Tips

  • Use simple, clear field names (name, title, price)
  • Set multi-value separator for combined fields
  • Create multiple groups for batch processing
  • Groups apply cyclically if you have more components than groups
